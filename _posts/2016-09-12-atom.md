### おすすめパッケージ
```
・autocomplete-plus
予測変換してくれる

・color-picker
色合いを確認できるパッケージ

・highlight-selected
洗濯したテキストと同じものをハイライトする

・minimap
sublimeのような使い方ができる

・minimap-autohide
自動で非表示にしてくれる

・minimap-bookmarks
どこにブックマークがあるかわかりやすくしてくれる

・project-manager
プロジェクトの管理をしてくれる

・linter
プログラムのエラーをリアルタイムに表示してくれる(対応のパッケージインストールが必要)

・sublime-style-column-selection
optionキーを押しながら短形選択できる

・git-plus
atomでgitの操作ができる

・tool-bar
atomにツールバーを追加する

・Flex Tool Bar
ツールバーをカスタマイズできる

・Remote-FTP
atomから直接FTP接続する

・docblockr
コメントを記載する際に便利

・vim-mode-plus
atomをvimっぽく使う

・vim-mode-clipbord-plus
OSのクリップボードと連携してくれる

・vim-mode-visual-block
短形選択できるようにする
```

### git-plusの使い方

```
初めに`user.email`と`user.name`とSSHの鍵認証の設定する必要がある
http://qiita.com/katsukii/items/9fd5bbe822904d7cdd0a


1.git-plus用のコマンドパレットを開く
cmd+shift+H

2.行いたい操作を選択

・便利な機能
addとcommitとpushをまとめてやってくれる

cmd+shift+a p

でOK

```


### よく使うショートカット
```
・コマンドパレット表示
cmd+shift+p

・設定表示
cmd+,

・ツリー表示・非表示

・新規ファイル作成
ツリー上で
shift+a

・ブックマーク登録
cmd+F2

・ブックマーク解除
ctrl+F2

・一行選択
cmd+l

・1行削除
ctrl-shift-K

・末尾までカット
ctrl-K

・単語の先頭・末尾まで削除
alt-H / alt-D

・カーソル追加
cmd-click

・複数行の選択からマルチカーソルへ
cmd-shift-L

・次の同じ単語を追加選択
cmd-D

・同じ単語を全選択	
ctrl-cmd-G
```
